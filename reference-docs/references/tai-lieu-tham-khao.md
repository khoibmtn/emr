# Danh mục tài liệu tham khảo mẫu — Định dạng Vancouver

## Hướng dẫn sử dụng

- Đánh số theo thứ tự **xuất hiện lần đầu** trong văn bản báo cáo
- Số trích dẫn đặt trong **ngoặc vuông**: [1], [2], [1,2]
- Tên tài liệu **in đậm**, thông tin xuất bản *in nghiêng*
- Nếu có nhiều tác giả: liệt kê 6 tác giả đầu, sau đó thêm "và cộng sự"

## Hướng dẫn sử dụng trong script Node.js

```javascript
// Cấu trúc 1 tài liệu tham khảo trong mảng refs[]
{
  n: "[1]",         // số trích dẫn
  parts: [
    {t: "Tên tác giả. "},                           // tác giả — không in đậm, không in nghiêng
    {t: "Tên tài liệu.", b: true},                   // tên tài liệu — in đậm (b:true)
    {t: " Thông tin xuất bản.", i: true},            // xuất bản — in nghiêng (i:true)
  ]
}

// Render thành Paragraph với hanging indent (chuẩn Vancouver)
new Paragraph({
  spacing: { before: 60, after: 100 },
  indent:  { left: 540, hanging: 540 },
  children: [
    txt("[1] ", {bold: true, size: 20, color: C.navy}),
    txt("Tên tác giả. ", {size: 20}),
    txt("Tên tài liệu.", {bold: true, size: 20}),
    txt(" Thông tin xuất bản.", {italic: true, size: 20}),
  ]
})
```
